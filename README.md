# 🤖 Cyris

**Cyris** is a modular AI desktop assistant built in Python. It is designed to automate everyday desktop tasks, manage workflows, interact with applications, perform internet searches, remember user information, and continuously evolve into a fully featured personal AI assistant.

---

# ✨ Features

## 🎤 Voice Assistant

* Wake word detection
* Speech recognition
* Text-to-speech responses

## 💻 Desktop Automation

* Open Chrome
* Open Brave
* Open VS Code
* Open Notepad
* Open Calculator
* Open ChatGPT
* Open Claude
* Open Antigravity
* Open Spotify
* Launch other installed applications

## 🌐 Internet Module

* Google Search
* YouTube Search
* GitHub Search

## 🧠 Memory

* Store custom memories
* Recall saved memories

## ⚙️ Workflows

* Coding Mode
* Expandable workflow architecture

## 📊 System Monitoring

* Battery
* RAM
* CPU
* Storage

## 📸 Vision

* Screenshot capture
* Vision command handler
* Foundation for future AI vision capabilities

## 📝 Logging

* Activity logging
* Command logging
* Memory logging
* Workflow logging
* Internet logging
